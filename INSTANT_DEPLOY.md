## **OPTION 1: Deploy with Vercel Web UI (Fastest - 3 minutes)**

### Step 1: Push to GitHub
```bash
cd TechHackathon
git remote add origin https://github.com/YOUR_USERNAME/PathLens.git
git branch -M main
git push -u origin main
```

### Step 2: Deploy on Vercel (No terminal needed)
1. Go to https://vercel.com/new
2. Click "Import Git Repository"
3. Paste: `https://github.com/YOUR_USERNAME/PathLens`
4. Click "Import"
5. Click "Deploy"
6. **DONE** - Your live URL appears in 2 minutes

Live URL will be: `https://pathlens-xxxxx.vercel.app`

---

## **OPTION 2: Deploy with Vercel CLI (If you have time)**

```bash
npm install -g vercel
cd TechHackathon
vercel
```

Follow prompts, accept defaults. Done in 3 minutes.

## **OPTION 3: Use Netlify (Alternative, same speed)**

1. Go to https://netlify.com
2. Click "New site from Git"
3. Connect GitHub
4. Select PathLens repo
5. Click "Deploy"

Done in 2 minutes.
