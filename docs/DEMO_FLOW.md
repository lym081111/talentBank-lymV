# Demo Flow

PathLens is designed for a short Stage 1 judge walkthrough, but the flow now behaves like a Career OS jobsite shell instead of only a demo gallery.

## Recommended Path

1. Open the live prototype: https://talentbank-lymv-career-os.vercel.app
2. Create a local prototype account or sign in.
3. Land on Career OS Home and notice the Starter Kit coverage strip.
4. Open **Search jobs** and test keyword, market, and work-mode filters.
5. Save a mock application, then open the application pack.
6. Open **Build profile** to enter student evidence, or use **Reset sample** to load a seeded student profile.
7. Continue through the main flow:
   - Profile Builder
   - Evidence Passport
   - Skill Extraction
   - Readiness Dashboard
   - Applications and Paths
   - Employer Brief
   - University Intervention Board

## What Judges Should Notice

- PathLens is **Career OS + Challenge Module**: the Career OS layer is login/register, profile/resume building, job listings, keyword search, matching, applications, candidate dashboard, employer dashboard, and university readout.
- The prototype is focused on **Universities Module 03: Adaptive Readiness Profile**.
- PathLens is **not a prediction engine**. It makes readiness visible from student evidence.
- Skills are extracted with source evidence, matched phrase, and confidence.
- Readiness scoring is deterministic and auditable.
- AI is used for explanation only: transparent evidence extraction + AI-assisted explanation.
- University view recommends interventions based on top readiness gaps.

## Starter Kit Coverage

- Profile and resume builder: Proof Passport profile and evidence editor.
- Job listings: searchable mock roles with salary bands, work mode, and required skills.
- Keyword and job search: filters for title, company, skill, market, and work mode.
- Job matching: evidence-backed role fit with blocker and next action.
- Candidate dashboard: Talent View readiness, skill extraction, gaps, and application next steps.
- Employer dashboard: lightweight role-specific shortlist brief and interview probes.
- Applications: saved mock applications plus evidence-backed claims, blockers, and talking points.
- Search and discovery: employer role selector changes the best-fit candidate ranking.
- Sign up and register: local prototype login/register/logout.

## Build Your Own Path

Use **Build profile** from Career OS Home to start with a blank profile. Use **Reset sample** only when a judge wants to inspect seeded evidence quickly.
